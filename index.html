My Work

My Currency Converter is a straightforward web app built with HTML, CSS, and JavaScript. It uses currency values from a GitHub repository to convert different currencies. Hosted on GitHub Pages, this tool offers a quick and easy way to perform currency conversions without relying on real-time exchange rates.

Our team, Coding Dragons, participated in a hackathon organized by ISRO in collaboration with Hack2skill, where we developed an innovative voice-enabled web map interface aimed at enhancing user interaction and accessibility. Our project integrates the SpeechKITT library, allowing users to control the map using voice commands for various tasks such as searching for locations, toggling map layers, placing markers, and navigating. This feature-rich solution uses the Leaflet GIS library to efficiently manage and display geospatial data, providing users with dynamic and interactive experiences.

To offer detailed visualizations, we incorporated WMS layers from Bhuvan and Bhoonidhi, enabling the display of glaciers, lake water bodies, and satellite views through Google Maps. The backend is built using Python and Flask, ensuring scalability and reliability, while the frontend leverages JavaScript and Leaflet for a responsive and user-friendly interface. Additionally, we enhanced accessibility by integrating UserWay's Accessibility Menu, which allows users to adjust features like contrast, alignment, and saturation to suit their needs.

Our solution stands out for its use of the SpeechKITT library by annyang, which offers faster and more stable voice command recognition compared to other speech-to-text solutions. This voice-enabled mapping system provides an innovative and efficient tool for interacting with geospatial data, making it more accessible and easy to use.
